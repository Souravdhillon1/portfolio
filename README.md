# 🌐 Personal Portfolio Website

This is my personal portfolio website built using **React.js** and **Tailwind CSS**. It presents a clean, modern, and responsive design to showcase my profile, skills, services, and provide a way for people to contact me.

## 📁 Sections

- **🏠 Home:** Brief intro with name, tagline, and call-to-action.
- **👤 About:** Details about who I am, background, and interests.
- **🛠 Skills:** List of technical skills and tools I use.
- **🧰 Services:** Services I offer (e.g., Web Development, UI Design).
- **📞 Contact:** Contact form powered by Formspree and social links.

## 🚀 Tech Stack

- **Framework:** React.js
- **Styling:** Tailwind CSS
- **Contact Form:** Formspree
- **Deployment:** GitHub Pages / Vercel / Netlify
